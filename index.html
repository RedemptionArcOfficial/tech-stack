Project Tech Stack

Agent Core

AI Models

Gemini Pro, Vision, STT

Gemini Pro excels in text generation, Vision extracts text from images, and STT analyzes student audio for reading evaluation

Tool Functions

Python/Node.js

Implements custom tools like worksheet generators and lesson planners in Python/Node.js for flexible, scalable workflows

Frontend UI

React.js + Tailwind

Delivers a responsive, modern UI with reusable React components styled efficiently using Tailwind CSS for rapid development

Chat Interface

Gemini Chat UI

Enables intuitive teacher-agent interactions for tasks like weekly planning and generating quizzes from textbook images

Navigation

React Router + Tailwind

Features a NavBar with tabs for seamless access to attendance, planning, assessments, notifications, and other tools

Authentication

Firebase Auth

Ensures secure teacher authentication with seamless email and Google login integration for streamlined, safe access

Hosting

Firebase Hosting

Provides fast, free, and reliable hosting with Firebase Studio, optimized for hackathon scoring and efficient deployment

Database

Firebase Firestore

Efficiently manages teacher data, attendance records, syllabus plans, notifications, and detailed assessment results

Audio Storage

Firebase Storage

Securely stores student reading audio for evaluation, enabling detailed feedback within educational workflows

Notifications

Firebase Messaging

Delivers real-time notifications to students and parents about attendance, project updates, and critical events

Testing & Logs

Firebase Emulator

Facilitates local debugging of cloud functions and thorough testing of agent responses to ensure reliability
